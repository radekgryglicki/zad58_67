Kod Wewnętrzny

  1. Gdzie jest wstawiany

    2. Wstawia się zawsze w części nagłówkowej dokumentu (pomiędzy znacznikami  < head >  a  < / head > ).
    Można go zastosować, gdy elementy które pragniemy poddać formatowaniu,
    występują wielokrotnie na stronie i wszystkim chcemy nadać takie same atrybuty (inne niż domyślne).

  2. Składnia teoretyczna

    3.  < head > 
            (...)
        < style type="text/css">
       /* < ! [ CDATA [ */
         selektor1  { cecha: wartość; cecha2: wartość2... }
         selektor2  { cecha: wartość; cecha2: wartość2... }
          (...)
        /* ]]> */
        < / style >
            (...)
     < / head > 

  3. Jakie mogą być selektory,

    4. Selektorem może być praktycznie dowolny znacznik,
    np. p (akapit), h1 (tytuł), td (komórka tabeli) i inne.
    To właśnie elementom, które znajdują się w kodzie źródłowym,
    pomiędzy tymi znacznikami, nadajemy atrybuty formatowania opisane w arkuszu.

  4. Jakie mogą być cechy i wartości.

    5. Jako "cecha" (w deklaracji stylu - powyżej)
    należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi
    (opisane w kolejnych rozdziałach).
    Natomiast jako wyraz "wartość" wpisujemy dokładną wartość atrybutu.
    Zwróć uwagę, że jednemu selektorowi możemy nadać kilka atrybutów (cech).
    Są one wtedy rozdzielone średnikami
